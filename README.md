## ✨ Features

### 🎬 Cinematic Terminal Preloader
A custom boot sequence with glitchy hex visuals and a dramatic reveal.

### 🌈 Dynamic Hue‑Shift Engine
Scroll-linked `--hue-shift` rotates the neon palette smoothly across sections.

### 🌀 Interactive WebGL Background (Three.js)
A responsive 3D particle mesh that reacts to cursor movement and resizes cleanly.

### 🧊 Glassmorphism + Depth System
Layered blur, grid composition, and SVG/noise textures for premium depth.

### ⚡ Hardware‑Accelerated Animations (No heavy frameworks)
Stagger reveals, clip-path navigation transitions, magnetic hover feel, smooth easing.

### 🧩 Bento Grid Layout
Modern asymmetrical bento-style layout for skills, work, and highlights.

## 🧰 Tech Stack

**Structure**
- Semantic HTML5

**Styling**
- CSS3 (custom properties, grid systems, blur/depth)

- Vanilla JavaScript (ES6+)
- IntersectionObserver API

**3D / Visuals**
- Three.js (WebGL)

**Typography**
- Syne (Display)

## 🚀 Run Locally

No complicated build pipeline required.

### 1) Clone
```bash
git clone https://github.com/Abhi-2636/portfolio.git
```

### 2) Enter folder
```bash
cd portfolio
```

### 3) Start
- Open `index.html` in your browser  
**OR (recommended)** run a local server:
- VS Code → install **Live Server** → Right click `index.html` → “Open with Live Server”
